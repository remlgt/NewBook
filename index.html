Rémi LOIGEROT

Développeur web junior

Jeune développeur, passioné par le web depuis longtemps ( conception, streaming, technologies, etc ). Avec quelques projets web à mon actif, je cherche à développer mon expérience professionelle. Sortant de l'école O'clock, j'ai réalisé plusieurs applications plus ou moins compliquées.
Ayant grandis avec un PC j'ai toujours passé beaucoup de temps dessus, afin de toujours découvrir de nouvelles technologie, de nouveaux jeux, ou de nouveau contenu. Je suis donc très à l'aise avec les outils informatiques en général.

Mes projets

COMPÉTENCES

Programmation web

Durant ma formation j'ai eu l'occasion de pratiquer les langages courant utilisés dans sur le web soit : HTML, CSS, JavaScript, PHP et MySQL. J'ai ensuite choisis de suivre une spécialité sur le framework Symfony avec des langages comme YAML et TWIG afin de me familiariser avec le back-office. Enfin sur des projets personnels j'ai pu utiliser des langages comme PUG, SASS ou encore débuter sur des Framework comme Three.Js ou React afin de m'améliorer en Front-end. J'ai aussi eu l'occasion d'utliser des bibliothèques comme Bootstrap, Bulma ou Tailwind.

Outils de bureautique & Vidéo

J'ai l'habitude de travailler sur différent OS tels que Windows, MacOs ou Linux Ubuntu ce qui peut-être un plus. Je développe sur des IDE comme ATOM ou Visual Code Studio la plus part du temps. J'ai également une bonne maitrise des logiciels de bureautique tel que World, Google Doc et même Trello afin d'organiser la construction d'un projet. La création ou la modification de contenu n'est pas un soucis pour un moi, j'ai plusieurs connaissances sur Photoshop, Adobe Premiere ou même OBS Studio.

Environnement web

J'utilise régulièrement l'outil Git/GitHub afin de gérer mes projets mais également de les partager, comme sur le site Codepen sur lequel je réalise les challenges hebdomadaire dans le but de m'améliorer. J'ai également passé la certification Opquast sur les bonnes pratiques à adopter sur le web tel que l'accessibilité pour les personnes en situation de handicap. La mise en place du responsive et des règles d'UI et d'UX sont importantes pour moi afin d'obtenir une application ergonomique à sa conception et dans son usage. J'ai également appris à travailler avec le concept de Sprint tiré de la méthode AGILE.
